Method: Geometric Brownian Motion, Euler-Maruyama discretization, antithetic variates. American options use Longstaff-Schwartz (LSM) least-squares regression for early exercise. Greeks are Black-Scholes analytical. Python runs in-browser via PyScript (Pyodide/WebAssembly) — no server required.
